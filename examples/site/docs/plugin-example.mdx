# PluginExample & Citation

This page exercises the `PluginExample` and `Citation` components moved into
`@jspsych/docusaurus-theme` in phase 4. `PluginExample`'s defaults (jsPsych
version, the "Run demo" controls plugin, and the harness asset paths) come
from `themeConfig.jspsych.pluginExample` — this site doesn't set that, so it
should fall back to the theme's built-in defaults.

## PluginExample

<PluginExample
  summary="Displaying a trial until the participant responds"
  dependencies={["@jspsych/plugin-html-keyboard-response@2.1.0"]}
  code={`const trial = {
  type: jsPsychHtmlKeyboardResponse,
  stimulus: '<p style="font-size:48px; color:green;">BLUE</p>',
  choices: ['r', 'g', 'b'],
  prompt: '<p>Is the ink color (r)ed, (g)reen, or (b)lue?</p>'
};

const timeline = [trial];`}
/>
